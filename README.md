# acb_draggable_form

A Python package that provides a draggable form using tkinter.

## Installation

You can install this package via pip:

```shell
pip install acb_draggable_form


Usage
Here's an example of how to use the draggable form in your Python code:

from your_package import DraggableForm
import tkinter as tk

# Create a tkinter window
root = tk.Tk()

# Create an instance of the DraggableForm
form = DraggableForm(root)
form.pack()

# Start the tkinter event loop
root.mainloop()
